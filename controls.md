# BASIC CONTROLS

## MOVEMENT

Walk forwards
: Gamepad: `Right`


: Keyboard: `D/Right`

Walk backwards
: Gamepad: `Left`

: Keyboard: `A/Left`

Jump
: Gamepad: `A`

: Keyboard: `Space`

## ACTIONS

Shoot a projectile
: Gamepad: `B`

: Keyboard: `Enter / Return`
